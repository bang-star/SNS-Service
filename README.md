# SNS-Service

## 개발 환경

* Intellij IDEA Ultimate 2022.1.1 ~ 2022.1.3
* Java 17
* Gradle 7.4.1
* Spring Boot 2.7.5

## 기술 세부 스택

Spring Boot

* Spring Web
* Spring Data JPA
* Spring Security
* Hibernate-types
* PostgresSQL Driver
* Lombok
* Apache Kafka
* jwt
* Redis

그 외

* CloudType
